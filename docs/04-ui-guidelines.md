# Guia visual de Coruscant Shell

## Direccion visual

- Estetica limpia, translucida, jerarquia fuerte.
- Inspiracion iOS sin copiar activos propietarios.
- Priorizar legibilidad, espaciado y respuesta tactil.

## Tokens base

- Radio principal: `22px`
- Sombra elevacion: `0 10px 35px rgba(0,0,0,.28)`
- Duracion animacion base: `220ms`
- Curva: `cubic-bezier(.2,.8,.2,1)`

## Componentes MVP

1. Lock screen
2. Home grid
3. Dock
4. Quick settings
5. App cards (switcher)

## Reglas UX

- Objetivos tactiles >= 44px.
- Contraste minimo AA.
- Estados claros: normal, activo, disabled, error.
- Gestos consistentes (swipe up/down/left/right).
